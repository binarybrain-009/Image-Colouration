Problem Statement

This project aims to experiment with various deep learning models to understand their performance in automatic image coloration tasks. By comparing different architectures, the study seeks to analyze how each model addresses the challenges of transforming grayscale images into realistic color versions. The goal is to explore and study the theoretical workings of these models in relation to image coloration and finalise on an architecture that is best suited to image coloration tasks.

Approach

The approach involves building a convolutional neural network (CNN) that learns to predict the color information from grayscale images. The image coloration task will be formulated as a supervised learning problem, where grayscale images, consisting of L* channel, are the inputs, and the corresponding color images are the targets which consists of two additional channels, A* and B*.
  1. Data Representation: Convert images to their corresponding grayscale image to be used as input to the model and the original image as an output.
  2. Model Architectures: For our first set of experiments, we intend to use a simple CNN based encoder-decoder architecture. Following that we intend to use U-Net with a VGG backbone, where the encoder learns latent grayscale features and the decoder reconstructs the color information. The extracted features will be fed into the decoder to predict the color channels, A* and B*. Finally, we intend to use GANs with the U-net model as the Generator.
  3. Loss Function: Mean Squared Error (MSE) between predicted and actual images to measure pixel-wise accuracy. We also intend to incorporate adversarial loss for GAN networks and experiment with them.
  4. Optimization: The model will be trained using the Adam optimizer with learning rate scheduling to adjust learning rates dynamically.

Plans for experiments

  • Datasets to be Used: ImageNet-tiny, CIFAR-10

  • Things we will be implementing:
    • Data loading
    • Colorization Model functions -> defining the architectures
    • Preprocessing
    • Fine tuning the architectures based on the datasets
    • Evaluation code:
      • Loss Function:Mean Squared Error (MSE) between predicted and actual A and B channels to measure pixel-wise accuracy.
      • Perceptual Loss: Arch like ResNet, VGG etc will be used to compute a perceptual loss that ensures colorized images look realistic based on higher-level image features.
      • Optimization: The model will be trained using the Adam optimizer with learning rate scheduling to adjust learning rates dynamically.

  • Code that will be borrowed: Pre-trained model architecture code for U-Net and GANs (particularly pix2pix architecture).

  • Definition for success of project:
    • As we move from CNN to U-net to GAN, the quality of image should improve especially with the incorporation of adversarial training in GANs.
    • To quantify the metrics we will look at MSE, IOU (Intersection over Union)

  • List of experiments to be performed:
    • Training a simple CNN encoder-decoder
    • Training a UNET architecture without adversarial training
    • Train a GAN with adversarial training

  • Expectation from experiments or what is uncertain about the potential of outcomes:
    The model is expected to produce realistic colorizations for a wide variety of images. However, context-sensitive colorization (e.g., predicting specific object colors) may pose challenges since we take one-channel input image to predict 2 additional channels.
