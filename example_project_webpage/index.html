Problem Statement

This project aims to experiment with various deep learning models to understand their performance in automatic image coloration tasks. By comparing different architectures, the study seeks to analyze how each model addresses the challenges of transforming grayscale images into realistic color versions. The goal is to explore and study the theoretical workings of these models in relation to image coloration.
